# 利用用户行为数据

## 基于邻域的算法

### 基于用户的协同滤波算法

#### 数据集

使用MovieLens数据集（6000用户+4000部电影）进行测试

#### 实验

**UserCF算法**和**UserIIF算法**

参数：

- M: 进行多少次实验（9）

- K: TopK相似用户个数（10）

- N: TopN推荐物品的个数（80）
- 进行两次交叉验证，取平均值

实验结果

|         | Precision | Recall | Coverage | Popularity | Time/s |
| :-----: | :-------: | :----: | :------: | :--------: | :----: |
| UserCF  |  0.2152   | 0.1281 |  0.2073  |   7.3057   | 238.28 |
| UserIIF |  0.2181   | 0.1298 |  0.2182  |   7.2760   | 553.30 |
